# Debate Arena 🎙️

A multiplayer AI debate app — two people join a room, each states their side, 
and an AI argues for each person. A judge AI gives the final verdict.

## How it works
1. Person A creates a room and gets a shareable link
2. Person B opens the link and joins
3. Both type their side of the argument
4. AIs debate in real time — both screens update live
5. Judge AI gives the final verdict

## Tech
- Vanilla HTML/CSS/JS — no framework
- Firebase Realtime Database — multiplayer rooms
- Anthropic Claude API — AI debate engine
- Hosted on GitHub Pages — free

## Run it
Just open the link. No install, no setup.
Bring your own API key (free at console.anthropic.com).
